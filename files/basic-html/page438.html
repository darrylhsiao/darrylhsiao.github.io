3.4  植生調查分析與應用



                  3.4.1  重要性與種歧異度之計算

                     一、重要值指數之計算：

                          植物社會介量以重要值指數值(important value index, IVI)表示。將植物社會

                     分成上下兩層(喬木層和地被層)，計算各種植物在各樣區中之密度、頻度及優勢

                     度，再轉換成相對值，上層植物社會重要值即三者相對值之總和，下層植物社會
                     重要值即相對頻度和相對優勢度之總和，其意義代表某植物在林分樣區中所佔

                     有之重要性。各計算式如下：



                                                    某種植物樹冠投影面積(或胸高斷面積)之總和
